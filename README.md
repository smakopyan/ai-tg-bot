## Описание проекта

Telegram-бот, интегрирующий различные AI-модели для генерации текста и изображений. Бот поддерживает несколько языковых моделей и предоставляет удобный интерфейс для взаимодействия с ними.

## Архитектура

### Основные компоненты:

1. **main.py** - точка входа, инициализирует бота и диспетчер
2. **handlers/** - обработчики событий:
   - `commands.py` - обработка команд (/start, /menu, /help)
   - `messages.py` - обработка текстовых сообщений
   - `callbacks.py` - обработка callback'ов от inline-кнопок
3. **services/** - бизнес-логика:
   - `ai_service.py` - интеграция с AI-провайдерами (OpenAI, Groq, Mistral, DeepSeek, GigaChat)
   - `history_manager.py` - управление историей диалогов через Yandex Object Storage
4. **utils/split_text.py** - утилита для разбивки длинных ответов

### Поддерживаемые AI-модели:
- Mistral (`mistral-large-latest`)
- DeepSeek (`deepseek-chat`) - (временно недоступно)
- GigaChat (`GigaChat-Pro`)
- OpenAI (через Groq API) - (временно недоступно)

### Функциональность:
- Генерация текста на основе диалоговой истории
- Выбор AI-модели через inline-меню
- Хранение истории диалогов в Yandex Object Storage
- Поддержка команд для навигации
- Партнёрская программа и опции поддержки

### Технологический стек:
- Python 3.8+ (3.12)
- Aiogram (фреймворк для Telegram-ботов)
- Boto3 (для работы с S3-совместимым хранилищем)
- Various AI SDK (OpenAI, Groq, Mistral, etc.)
- Yandex Object Storage для хранения истории

### Конфигурация:
Бот использует переменные окружения для конфигурации:
- `BOT_TOKEN` - токен Telegram бота
- AI API ключи для каждого провайдера
- Yandex Object Storage credentials
- Параметры истории диалогов

Этот проект представляет собой гибкую и масштабируемую архитектуру для создания AI-ассистента в Telegram с поддержкой множественных провайдеров и удобным пользовательским интерфейсом.
